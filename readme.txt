== Description ==
HasOffers is a Seattle based company offering performance tracking or affiliate marketing campaigns tracking software to businesses. With this software, businesses can track and manage their own affiliate programs. 

The plugin enables businesses to get connected with HasOffers and automatically get their end usersâ€™ data forwarded to the performance tracking software.  The plugin allows the business users to enter their credentials from the backend which will be automatically forwarded to HasOffers.  

== Installation ==
1. Go to Plugin > Add New
2. Type in the name of the WordPress Plugin or descriptive keyword, author, or tag \"hasoffers\" in Search Plugins box.
3. Find the HasOffers Plugin and install.
      1. Click Details for more information about the Plugin and instructions you may wish to print or save to help setup the Plugin.
      2.Click Install Now to install the WordPress Plugin. 
4. The resulting installation screen will list the installation as successful or note any problems during the install.
5. If successful, click Activate Plugin to activate it, or Return to Plugin Installer for further actions.
6. Find a shortcode [hasoffer-form].
